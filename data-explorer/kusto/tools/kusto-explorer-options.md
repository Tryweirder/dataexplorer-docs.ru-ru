# <a name="kusto-explorer-options"></a>Параметры Kusto Explorer

В следующих таблицах описаны варианты настройки поведения Kusto.Explorer из диалогового окна > **«Инструменты параметры».** **Tools**

![«Инструменты» > »Варианты»](\images\Kusto-Tools-Explorer-Options\tools-options.png)

## <a name="general-settings"></a>Общие параметры

| Параметр | Описание |
|---------|--------------|
| Режим инструмента | Включает бета-, альфа- и экспериментальные функции применения. Значение по умолчанию — none (Отсутствует).|
| Расширенная доступность | При включении приложение отправляет события доступности, используемые инструментами доступности. Может повлиять на производительность при включении. По умолчанию отключен. |
| Разрешить отправку телеметрических данных | При включении приложение отправляет данные телеметрии при возникновении ошибок или сбоях приложения. |
| Приветственное сообщение | При включении приложение показывает приветственное сообщение на старте. По умолчанию включено.|
| Тема показа | Цветовая гамма приложения: светлая или темная.|
  
## <a name="query-editor"></a>Редактор запросов

| Параметр | Описание |
|---------|--------------|
| Запросы авто-сохранения | При включении приложение автоматически сохраняет открытые документы. Изменение этой настройки требует перезагрузки приложения для вхлечения в силу. Включен по умолчанию.|
| Отслеживание изменений | При включении приложение отслеживает изменения, внесенные в запрос скриптов на диске другими приложениями. Если скрипт запроса изменен внешне, вы будете уведомлены и вызваны для его перезагрузки. Изменение этой настройки требует перезагрузки приложения. Включен по умолчанию.|
| Используйте сеансы edit | При включении каждый процесс подачи заявки имеет свой собственный набор документов. Отключено по умолчанию.|
| Размер шрифта | Размер шрифта, используемый в редакторе запроса.|
| Выберите фон команды | Цвет фона, используемый для выделения выбранной в настоящее время команды.|
| Замена вкладок пробелами | При включении вкладки автоматически заменяются пробелами.|
| Инъекция параметра функции отсутсвия | При включении из буфера обмена отключается впрыск параметра функции.|
| Выгоняемая инъекция параметра запроса | При включении впрыск параметра запроса отключается.|
| Отключить триггеры выполнения запроса, за исключением F5 | При включении только ключ F5 запускает запросы для запуска.|
| Показать справку внутри приложения | При включении темы справки отображаются внутри приложения. При открытии в браузере темы справки отключены.|
| Ограничение длины параметра запроса | Максимальная длина строки, которая может быть использована в качестве параметра запроса. Установка этого значения на более чем 128K может вызвать проблемы с производительностью. По умолчанию 64K.|

## <a name="intellisense"></a>IntelliSense

| Параметр | Описание |
|---------|--------------|
| Версия IntelliSense | Используется версия двигателя IntelliSense. *V1* является классическим двигателем. *V2* - это современный двигатель. По умолчанию *является V2*. |
| IntelliSense: Команды управления | Версия IntelliSense для команд управления. *V1* является классическим двигателем. *V2* - это современный двигатель. По умолчанию *является V2*. | 
| IntelliSense | Включает или отсвадает IntelliSense. По умолчанию включено.|
| Выделение синтаксиса | Позволяет или отсваивает выделение синтаксиса. По умолчанию включено.|
| Советы по инструментам | Включает или отсвачивает наконечники инструментов, которые появляются, когда мышь парит над областями выбранного запроса. По умолчанию включено.|

## <a name="formatter"></a>Formatter

| Параметр | Описание |
|---------|--------------|
| Версия | Версия formatter, используемая при применении инструмента Prettify Query к текущему запросу. *V1* является классическим formatter. *V2* является современным formatter. По умолчанию *является V2*.|
| Indentation; | Количество мест для отступных элементов.|
| Функциональные скобки | Размещение функциональных скобок. *Вертикальные* места открываются и закрывают фигурные скобки на новых линиях. *Горизонтальная* оставляет открытую скобку на своей первоначальной линии и помещает близкую скобку на новую линию. *Никто не* оставляет все скобки, как есть.|
| Оператор трубопроводов | Размещение символа трубы (бара), который существует между операторами табеля-запросов. *New Line* помещает все символы труб на новую линию. *Smart* помещает все символы труб на новую строку, если запрос уже охватывает более одной строки. *Никто не* оставляет все символы трубы, как есть.|
| Точка с запятой | Размещение запятых, которые заканчивают заявления запроса. *Новая линия* помещает запятые на новую линию с отступом. *Smart* помещает запятые на новую линию, если само утверждение охватывает более одной строки.  *Никто не* оставляет запятые как есть.
| Вставить отсутствующие Синтаксис | Добавляет отсутствующие знаки препинания, как (например, запятые и запятые), что вы получаете сообщения об ошибках.|

## <a name="results-viewer"></a>Результаты Зритель

| Параметр | Описание |
|---------|--------------|
| Размер шрифта | Размер шрифта, используемый сеткой таблицы данных, где отображаются результаты запроса.|
| Цветовая гамма многообразия | Выбирает цветовую схему для форматирования строки на уровне многословности.|
| Скрыть пустые колонны | При включении пустые столбцы в представлении, показывающем данные, будут скрыты.  По умолчанию отключен.|
| Коллапс однозначных столбцов| При включении в представлении будут автоматически сворачиваться однозначные столбцы в представлении, отображающие данные. Столбцы с одними непустыми значениями будут отображаться как группы. По умолчанию отключен.|
| Результаты автоматической сортировки по столбику времени даты | При включении строки будут автоматически сортироваться по столбику времени даты. По умолчанию включено.|
| Столбец видимый диапазон | Максимальное количество символов, отображаемых в столбце. По умолчанию 2048.|
| Визуализируйте JSON как текст | При включении значения JSON визуализируются как текст. По умолчанию отключен.|
| Максимальный размер текстовых деталей | Максимальное количество символов, которые будут отображаться в подробной панели информации при двойном нажатии ячейки. По умолчанию 32KB.|

## <a name="connections"></a>Соединения

| Параметр | Описание |
|---------|--------------|
| Сортировка столбцов таблицы в алфавитном порядке | При включении столбцы, появляющиеся под узлами таблицы в панели соединений, будут сортироваться в алфавитном порядке.|
| Тайм-аут сервера запросов | Тайм-аут сервера для выполнения запроса.|
| Предупреждение о блокировке соединения | При включении приложение будет предупреждать о блокировке соединения при попытке переключения соединения. По умолчанию включено.|
| Ленивые исследования схемы | При включении панель соединений будет извивать и отображать схему базы данных только при расширении узла базы данных.|
| Отображить скрытые объекты системы | При включении скрытые объекты системы будут отображаться, если пользователь имеет соответствующие разрешения.|
| Слабая последовательность запросов | При включении для запросов будет использоваться слабая согласованность.|
| Версия спарзера Кусто | Версия парсера, используемая для выполнения запросов. *V1* является классическим парсером. *V2* - современный парсер. По умолчанию *является V1*.|
