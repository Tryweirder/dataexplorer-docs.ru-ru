# <a name="atan"></a>atan()

Возвращает угол, касательной которого является указанное [`tan()`](tanfunction.md)число (обратная операция ).

**Синтаксис**

`atan(`*X*`)`

**Аргументы**

* *x*: Реальное число.

**Возвращает**

* Значение дуги касательной`x`
