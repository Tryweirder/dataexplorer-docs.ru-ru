# <a name="gamma"></a>gamma()

Вычисляет [гамма-функцию](https://en.wikipedia.org/wiki/Gamma_function)

**Синтаксис**

`gamma(`*X*`)`

**Аргументы**

* *x*: Параметр для гамма-функции

**Возвращает**

* Гамма-функция x.
* Для вычислительной функции журнала-гаммы [см.](loggammafunction.md)
