# <a name="sin"></a>sin()

Возвращает функцию синуса.

## <a name="syntax"></a>Синтаксис

`sin(`*x*`)`

## <a name="arguments"></a>Аргументы

* *x*: вещественное число.

## <a name="returns"></a>Результаты

* Результат .`sin(x)`
