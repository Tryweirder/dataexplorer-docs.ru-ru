# <a name="cos"></a>cos()

Возвращает косинус.

## <a name="syntax"></a>Синтаксис

`cos(`*x*`)`

## <a name="arguments"></a>Аргументы

* *x*: вещественное число.

## <a name="returns"></a>Результаты

* Результат .`cos(x)`
