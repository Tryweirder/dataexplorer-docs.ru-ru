# <a name="new_guid"></a>new_guid()

Возвращает случайный идентификатор GUID (глобальный уникальный идентификатор).

```kusto
new_guid()
```

## <a name="syntax"></a>Синтаксис

* `new_guid()` — Создает новое значение типа `guid` .
