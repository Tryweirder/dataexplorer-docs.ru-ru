# <a name="tan"></a>tan()

Возвращает функцию тангенса.

## <a name="syntax"></a>Синтаксис

`tan(`*x*`)`

## <a name="arguments"></a>Аргументы

* *x*: вещественное число.

## <a name="returns"></a>Результаты

* Результат .`tan(x)`
