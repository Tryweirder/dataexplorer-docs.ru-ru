# <a name="pow"></a>pow()

Возвращает результат подъема к власти

**Синтаксис**

`pow(`*базовый* `,` *показатель*`)`

**Аргументы**

* *база*: Базовое значение.
* *экспонент*: Экспонентная стоимость.

**Возвращает**

Возвращает базу, поднятую на экспонент ате: база и экспонент.
