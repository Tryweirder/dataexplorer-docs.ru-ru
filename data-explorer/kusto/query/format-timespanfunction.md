# <a name="format_timespan"></a>format_timespan()

Форматы временной промежуток в соответствии с предоставленным форматом.

```kusto
format_timespan(time(14.02:03:04.12345), 'h:m:s.fffffff') == "2:3:4.1234500"
```

**Синтаксис**

`format_timespan(`*Формат* *тайм-шота* `,``)`

**Аргументы**

* `timespan`: значение типа `timespan`.
* `format`: строка для осечки формата, состоящая из одного или нескольких [элементов формата.](#supported-formats)

**Возвращает**

Строка с результатом формата.

|---------|-------|

print
