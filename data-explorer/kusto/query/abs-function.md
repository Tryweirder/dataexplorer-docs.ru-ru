# <a name="abs"></a>abs()

Рассчитывает абсолютное значение ввода.  

**Синтаксис**

`abs(`*X*`)`

**Аргументы**

* *x*: целый ряд или реальное число, или значение временной продолжительности.

**Возвращает**

* Абсолютное значение x.
