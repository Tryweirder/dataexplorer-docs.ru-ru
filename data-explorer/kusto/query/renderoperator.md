# <a name="render-operator"></a>Оператор render

Поручает агенту пользователя отображать результаты запроса определенным образом.

```kusto
range x from 0.0 to 2*pi() step 0.01 | extend y=sin(x) | render linechart
```

> [!NOTE]
> * Оператор рендера должен быть последним оператором в запросе и использоваться только с запросами, которые дают единый результат потока данных.
> * Оператор рендеринга не изменяет данные. Он вводит аннотацию ("Визуализация") в расширенные свойства результата. Аннотация содержит информацию, представленную оператором в запросе.
> * Интерпретация информации визуализации осуществляется агентом пользователя. Различные агенты (например, Kusto.Explorer,Kusto.WebExplorer) могут поддерживать различные визуализации.

**Синтаксис**

*T* `|` `with` `(` `=` *PropertyValue* `,` *PropertyName* *Visualization* Визуализация - НедвижимостьНедвижимостьСтоимость `render` `)`]

> [!NOTE]
> Модель данных оператора рендера смотрит на таблики, как будто она имеет три вида столбцов:
>
> * Столбец оси x `xcolumn` (указанный свойством).
> * Столбцы серии (любое количество столбцов, указанных `series` свойством). Для каждой записи комбинированные значения этих столбцов определяют одну серию, а диаграмма имеет столько же серий, сколько и отдельные комбинированные значения.
> * Столбцы оси y (любое количество `ycolumns` столбцов, указанных свойством).
  Для каждой записи серия имеет столько же измерений ("точки" на графике), сколько столбцов y-оси.

> [!TIP]
> 
> * `where`Используйте `summarize` `top` и ограничьте объем, который отображается.
> * Сортируйте данные, чтобы определить порядок оси x.
> * Агенты пользователей могут свободно «угадать» значение свойств, которые не указаны запросом. В частности, наличие «неинтересных» столбцов в схеме результата может привести к тому, что они угадывают неправильно. Попробуйте проецировать такие столбцы, когда это произойдет. 

**Пример**

```kusto
range x from -2 to 2 step 0.1
| extend sin = sin(x), cos = cos(x)
| extend x_sign = iif(x > 0, "x_pos", "x_neg")
| extend sum_sign = iif(sin + cos > 0, "sum_pos", "sum_neg")
| render linechart with  (ycolumns = sin, cos, series = x_sign, sum_sign)
```
