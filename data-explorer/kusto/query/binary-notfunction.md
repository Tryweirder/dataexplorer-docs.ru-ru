# <a name="binary_not"></a>binary_not()

Возвращает битное отрицание вхоза.

```kusto
binary_not(x)
```

**Синтаксис**

`binary_not(`*num1*`)`

**Аргументы**

* *num1*: числовая 

**Возвращает**

Возвращает логическую НЕ операцию на номер: num1.
