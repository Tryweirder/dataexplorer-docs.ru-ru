# <a name="cot"></a>cot()

Вычисляет тригонометрический котангенс указанного угла в радианах.

## <a name="syntax"></a>Синтаксис

`cot(`*x*`)`

## <a name="arguments"></a>Аргументы

* *x*: вещественное число.

## <a name="returns"></a>Результаты

* Значение функции котангенса для`x`
