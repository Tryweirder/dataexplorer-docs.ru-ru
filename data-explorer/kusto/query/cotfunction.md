# <a name="cot"></a>cot()

Рассчитывает тригонометрический котанген т.д. за указанный угол, в радианах.

**Синтаксис**

`cot(`*X*`)`

**Аргументы**

* *x*: Реальное число.

**Возвращает**

* Значение функции cotangent для`x`
